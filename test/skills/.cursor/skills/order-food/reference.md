## 查询要点

- **地点**：尽量具体到区域或地标（如「国贸」「公司楼下」），便于推荐附近餐厅与配送范围。
- **口味/菜系**：川菜、粤菜、日料、西餐、快餐、清淡、辣等，方便缩小范围。
- **人数与场景**：1 人外卖、2 人约会、多人聚餐，影响餐厅类型与菜量建议。
- **预算**：人均 30、50、100+ 等，在建议中标注人均区间。
- **忌口**：不吃辣、素食、过敏食材等，建议中避开或标注。

## 搜索关键词建议

- 外卖：`地点 + 外卖`、`地点 + 外卖 推荐`
- 餐厅：`地点 + 餐厅`、`地点 + 菜系 推荐`、`地点 + 人均 XX`
- 团购：`地点 + 团购`、`大众点评 地点`
