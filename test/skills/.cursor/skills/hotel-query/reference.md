# 酒店查询参考

## 常用平台（用户自行查价与预订）

| 平台 | 说明 | 链接示例 |
|------|------|----------|
| 携程 | 国内酒店、机票、套餐多 | 携程 App 或 ctrip.com |
| 美团 / 美团酒店 | 国内酒店、到店付、优惠多 | 美团 App 或 meituan.com |
| 飞猪 | 阿里系，国内+国际 | fliggy.com |
| Booking | 国际酒店多，部分国内 | booking.com |
| 去哪儿 | 国内酒店、比价 | qunar.com |

回复用户时可写：「实时价格与房态请以携程、美团、Booking 等平台为准，建议比价后下单。」

## 查询要点

- **城市/区域**：明确到城市或景区（如「杭州西湖附近」），便于推荐区域。
- **入住/退房日期**：有日期才能查价格与房态；若用户只说「周末」「下周」，可换算为具体日期或说明「以您选择日期为准」。
- **人数与房型**：2 人/家庭/大床/双床 等，影响房型与价格建议。
- **预算**：若用户提到预算（如「500 以内」），在建议中标注价格区间。

## 搜索关键词建议

- 中文：`城市 + 酒店 推荐`、`城市 + 入住日期 酒店 价格`
- 英文：`city + hotels`、`city + hotels check-in date`
- 可加上区域：`北京 王府井 酒店`、`上海 外滩 住宿`

## 注意

- 不替用户下单或绑定任何平台账号；仅提供查询结果与平台指引。
- 价格、房态、点评均以各平台实时为准，回复中注明「仅供参考」。
