# 天气生活建议规则

## 穿衣（按体感/最高温）

| 温度区间 | 建议 |
|----------|------|
| ≥ 30°C | 短袖短裤、防晒、多补水 |
| 24–29°C | 短袖/薄长袖，早晚可加薄外套 |
| 18–23°C | 长袖 + 薄外套，早晚略凉 |
| 10–17°C | 外套/针织衫，可加薄毛衣 |
| 0–9°C | 毛衣 + 厚外套或薄羽绒 |
| < 0°C | 羽绒服/厚大衣，帽子围巾手套 |

## 雨具

- 有降水（雨/雪）或降水概率 ≥ 50%：建议带伞或雨具。
- 降水概率 30–50%：可带折叠伞备用。
- 降水概率 < 30% 且无降水描述：一般无需带伞。

## 户外与运动

- **适宜**：晴/多云、体感 10–28°C、无强风、无恶劣预警 → 可推荐跑步、散步、骑行、户外活动。
- **注意**：高温（≥32°C）防中暑、补水；低温（<5°C）注意保暖、热身；有雨则改室内或雨后。
- **不推荐**：暴雨、雷暴、大风、霾/污染重时避免户外剧烈运动。

## 洗车与晾晒

- **适宜**：晴或多云、无降水、非大风 → 可建议洗车、晾晒。
- **不宜**：当日或次日有雨/雪、降水概率高 → 建议改期。
- **勉强**：阴天、降水概率 30–50% → 可晾晒但建议随时收。

## 紫外线

- 强：防晒霜 + 帽子/袖套，避免长时间暴晒。
- 中等：户外建议涂防晒。
- 弱：一般不需特别防护。
