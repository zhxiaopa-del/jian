# 桌面文件技能

本技能用于协助用户了解或整理桌面（Desktop）上的文件，包括列出文件、按类型/日期统计、给出整理建议或生成可执行的整理脚本。

## 何时使用

- 用户问「桌面有什么文件」「桌面有多少个文件」「桌面有哪些图片/文档」
- 用户问「怎么整理桌面」「桌面文件分类」「桌面清理」
- 用户需要按扩展名、修改日期等对桌面文件做统计或分组

## 桌面路径

- macOS：`~/Desktop` 或 `/Users/<用户名>/Desktop`
- 在脚本或命令中优先使用 `~/Desktop`，以便适配当前用户。

## 在项目根目录执行脚本

**仅列出并统计桌面文件（按扩展名分组，不移动任何文件）**：

```bash
python3 .cursor/skills/desktop-files/scripts/list_desktop.py
# 指定目录：python3 .cursor/skills/desktop-files/scripts/list_desktop.py ~/Desktop
```

输出：文件总数、按扩展名分组的数量与文件名列表。用户可根据输出决定如何整理。

## 整理建议（不跑脚本时）

若不便执行脚本，可根据用户需求口头给出建议，并遵循 [reference.md](reference.md) 中的命名与分类约定：

1. **先了解现状**：说明如何用 `ls -la ~/Desktop` 或 Finder 查看桌面文件。
2. **分类建议**：按类型建议文件夹名（如 文档、图片、压缩包、其他），见 reference。
3. **安全第一**：不要未经用户确认就移动或删除文件；若生成整理脚本，仅生成后交由用户审阅并自行执行。

## 生成整理脚本时

若用户明确要求「生成整理桌面的脚本」：

1. 先运行 `list_desktop.py` 或等价方式了解当前桌面文件类型。
2. 按 reference 的文件夹命名生成脚本（如 Python 或 Shell），脚本应：只移动文件到子文件夹、不删除、先创建目录再移动。
3. 在回复中说明「请审阅后再运行」，并把脚本保存路径写清楚。
