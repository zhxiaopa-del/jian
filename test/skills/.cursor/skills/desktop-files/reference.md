# 桌面文件整理参考

## 常用文件夹命名建议

| 类型       | 建议文件夹名 | 常见扩展名示例 |
|------------|--------------|----------------|
| 文档       | 文档         | .pdf, .doc, .docx, .txt, .md, .xlsx, .pptx |
| 图片       | 图片         | .jpg, .jpeg, .png, .gif, .webp, .heic |
| 视频       | 视频         | .mp4, .mov, .avi, .mkv |
| 音频       | 音频         | .mp3, .m4a, .wav |
| 压缩包     | 压缩包       | .zip, .rar, .7z |
| 安装包/安装 | 安装包       | .dmg, .pkg, .exe |
| 其他       | 其他         | 未匹配以上类型的文件 |

## 安全约定

- 不自动删除文件；若用户要求「删除某类」，需在脚本中加确认或注释说明。
- 移动前可先做「干跑」：只打印将要执行的操作，不实际移动，用户确认后再执行。
- 遇到同名文件时：可跳过或重命名（如 文件名_2.pdf），在脚本中写清策略。
